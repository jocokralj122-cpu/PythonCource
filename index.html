📘 Lesson 1: Python Basics

Welcome to the course! We're going to learn Python from the ground up. Not just what to type, but why you type it that way.

What is Python?

Python is a high-level, easy-to-read programming language created by Guido van Rossum, a Dutch programmer.

High-level means it's designed to be readable for humans — you don't need to manage memory or worry about low-level computer details. Python handles that for you.

Where to get it:

  • Computer: Download from python.org
  • Android: Install Pydroid from the Play Store
  • iOS: Install ACode from the App Store

What is Python Used For?

  • Sending emails
  • Creating games
  • Automating repetitive tasks
  • Web development
  • Data analysis
  • Artificial intelligence

Why Python?

Short answer: Python has a clear, simple syntax, and it comes with libraries that do the complex stuff for you.

A library is pre-written code that someone else made so you don't have to reinvent the wheel. Want to send an email? There's a library. Want to build a website? There's a library. You focus on your idea; Python handles the heavy lifting.

Strings

Think of a string as text. "John" is a string. "Hello world" is a string.

'John'
"John"

Why do quotes matter? Without quotes, Python would look for a variable with that name. If it doesn't exist, you'll get an error.

Printing

print("Hello")

Why the parentheses? print is a function — we'll cover functions later, but for now, know that () tells Python "run this thing."

💡 Exception: If you're printing a number, you don't need quotes. You can use them, but then you can't do math with it. 
print(5)      # prints 5 as a number
print("5")    # prints 5 as text — different!

Variables

Variables store information so you can use it later. Think of them like labeled boxes.

name = "John"    # text → needs quotes
age = 10         # number → no quotes

Why no spaces? Python reads your code word by word. my name would look like two separate things. Use underscores instead: my_name.

F-Strings

F-strings combine text with variables in a clean way.

name = "John"
age = 10
print(f"Hello {name}, your age is {age}")

Why the f? The f tells Python "this string has placeholders." Without it, Python would print {name} as literal text.

The time Library

import time
time.sleep(1)   # wait 1 second
print("Hello")

Why import? Python only loads the basics by default. When you need extra features, you import them.

🔄 Lesson 2: Loops

There are many types of loops, but for now we're going to focus on for loops.

The Structure of a for Loop

for i in range(1, 10):
    print(i)
  • for — starts the loop
  • i — variable holding the current value
  • in — connects variable to the range
  • range(1, 10) — numbers from 1 up to, but not including, 10
  • : — "the code below belongs to this loop"
  • Indentation (4 spaces) — Python knows which instructions to repeat

A Real Example

import time

for i in range(10, 15):
    time.sleep(1)
    print("Hello")
  • range(10, 15) gives: 10, 11, 12, 13, 14
  • The loop runs 5 times
  • Each time, it waits 1 second, then prints "Hello"
⚠️ Common confusion: Beginners often think range(10, 15) includes 15. It doesn't. Python stops before reaching the second number.

🎲 Lesson 3: Booleans & If Statements

Booleans

Think of them as the Yes/No switch. A boolean is either True or False.

triggered = True   # first letter must be uppercase

If Statements

If statements ask: "Is this boolean True?"

triggered = False

if triggered:
    print("Triggered")
💡 Pro tip: You could write if triggered == True:, but it's like asking "Is True equal to True?" Just write if triggered:. Cleaner, more Pythonic.

The not Keyword

What if you want to check if something is False?

triggered = False

if not triggered:
    print("Not triggered")

But not does more than booleans. It checks if something is "falsy":

name = ""
if not name:
    print("No name provided")  # Runs because empty string is falsy

count = 0
if not count:
    print("Nothing to count")  # Runs because 0 is falsy

items = []
if not items:
    print("Empty list")  # Runs because empty list is falsy

Falsy values: False, None, 0, "", [], {}, 0.0 — everything else is truthy.

Elif Statements

An if statement asks one question. What if you have multiple possibilities? That's where elif comes in — "if not that, then check this."

score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("F")
  • Checks first if. Is 85 >= 90? No.
  • Moves to elif. Is 85 >= 80? Yes. Prints "B".
  • Stops there. Doesn't check the other elifs.
